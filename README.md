# Proyecto 6: Computer Vision

Este proyecto simula un sistema de incio de sesión seguro mediante reconcimiento facial que al verificar la identidad del usuario carga automaticamente su perfil personalizado de Hogar Inteligente

1. Crear y activar el entorno virtual

python3 -m venv .venv

source .venv/bin/activate

2. instalar dependencias

pip install -r requirements.txt

4. Ejecutar

python3 main.py
